# inversao-de-nomes
Projeto: Inversão de Nomes em C#
📌 Descrição

Este programa em C# tem como objetivo ler quatro nomes digitados pelo usuário e exibi-los na ordem inversa.

⚙️ Funcionamento

O sistema solicita ao usuário a entrada de quatro nomes. Após isso, utiliza uma variável auxiliar para realizar a troca dos valores e inverter a sequência original.

🧠 Lógica Utilizada
Uso de variáveis do tipo string
Entrada de dados com Console.ReadLine()
Troca de valores com variável auxiliar
Saída de dados com Console.WriteLine()
▶️ Exemplo de Execução

Digite o nome#1 : João
Digite o nome#2 : Maria
Digite o nome#3 : Pedro
Digite o nome#4 : Ana

Nomes inseridos na sequência InverTIDA:
Ana
Pedro
Maria
João

Tecnologias Utilizadas
Linguagem: C#
Plataforma: .NET Console
👨‍💻 Autor

Projeto desenvolvido para fins educacionais.

O objetivo principal é praticar conceitos básicos de programação, como entrada e saída de dados, uso de variáveis e lógica de troca de valores.

🎯 Objetivo
Receber 4 nomes do usuário
Armazenar os nomes em variáveis
Inverter a ordem dos nomes
Exibir o resultado no console
⚙️ Como Funciona

O programa solicita ao usuário que digite quatro nomes. Em seguida, utiliza uma variável auxiliar para realizar a troca dos valores, invertendo a sequência original.

🔄 Lógica de Inversão
O primeiro nome troca com o quarto
O segundo nome troca com o terceiro
